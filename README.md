


# HomeFlow

[![Build and Test](https://github.com/mkinyon/HomeFlow/actions/workflows/build-test.yml/badge.svg)](https://github.com/mkinyon/HomeFlow/actions/workflows/build-test.yml)

**HomeFlow** is a Blazor Server application designed to help individuals and families organize everyday home life. From managing recipes and planning meals to building grocery lists, HomeFlow provides a streamlined, customizable platform for staying on top of it all.

---

## Features

- **Recipe Management**  
  Store, edit, and tag your favorite recipes with ingredients, directions and more.

- **Drag & Drop Grocery List Editor**  
  Create and organize grocery lists with an intuitive drag-and-drop interface.

- **Meal Planning**  
  Easy to use meal planning calendar with the ability to quickly add meals to a grocery list.

---

## Tech Stack

- **Frontend**: Blazor Server (.NET 9), MudBlazor  
- **Backend**: ASP.NET Core API  
- **Database**: SQLite, EF Core  
- **Architecture**: Vertical Slice Architecture  
- **Containerization Support**: Docker
